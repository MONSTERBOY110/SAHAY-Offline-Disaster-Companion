# SAHAY Knowledge Base

This directory holds the offline knowledge base for SAHAY (সহায়), an offline disaster-response assistant for cyclone- and flood-affected West Bengal. The bilingual (English + Bengali, bn-IN) first-aid, preparedness, and psychological-first-aid content is authored to reflect standard, widely accepted guidance in the style of IFRC/Indian Red Cross first-aid manuals, WHO/UNICEF public-health advice (ORS, cholera, water treatment), and NDMA/WBSDMA cyclone and flood safety materials; it deliberately avoids prescription medicines, invented dosages, and unverified folk remedies, and repeatedly directs users to call 112/108 and seek professional care when a network is available. Markdown files follow a strict structure, one `# Title` plus self-contained `## Section` blocks, so the BM25 RAG chunker can retrieve each section on its own. `helplines.json` contains only real, well-known Indian emergency numbers. **Disclaimer:** `shelters.json` is an illustrative sample dataset for demonstration only, the districts, blocks, and coordinates are plausible but the individual shelter entries are not an official registry; replace it with the official WBSDMA/NDMA shelter list before any production or field use, and have all medical content reviewed by qualified clinicians.
